Tech task created with:
- [Next.js](https://nextjs.org/)
and libraries:
- Json-server
- Axios
- Tailwind CSS

## Getting Started

1. Cd to project folder
2. Install dependancies

```bash
npm i
```

3. Run development server

```bash
npm run dev
```

4. Watch json-server

```bash
json-server db.json --port 5000
```
